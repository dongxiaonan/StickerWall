1 user can input sticker to the wall
2 user can add a tag to sticker by press enter key, also tag can set a color
3 user can filter sticker by tag
4 user can save data to localstorage
5 user can view basic date infomcation
6 user can see a random wisdom by some api ([https://github.com/gophergala/wisdom])
7 [optional] publish sticker wall to your github pages
8 [optional] user can change theme(background image) (image uploader without backend)
9 [optional] user can share current sticker wall with other guys by a link ([http://domain/sticker#uuidxxxexample])
10 [optional] multiple user can edit same sticker wall (firebase/websocket)
